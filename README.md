## 使用
```bash
  git clone https://github.com/HuangZhiming-the-Great/create-simple-react-.git
  cd create-simple-react-
  yarn
  yarn build
  yarn dev
```
接下来打来浏览器，输入：localhost:9000。就可以看到hello world!了。
